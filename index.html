Omega.

The next evolution of Epsilon.

Omega is an extension to Numworks' Epsilon, the OS that runs on their calculator, which brings many features to it. Omega is for people who want to add features to the calculator, but cannot because they have been rejected by Numworks (and for reasons that are 100% understandable!).

Symbolic computation

Symbolic computation has been removed from Epsilon in version 11.2. It has been reintroduced in Omega.

Periodic table

Inspired by the periodic table app of the TI83PCE, Omega's periodic table is clean and simple to use.

Periodic table

Inspired by the periodic table app of the TI83PCE, Omega's periodic table is clean and simple to use.

RPN

Omega supports inputting expression using Reverse Polish Notation.

And more!

Theme engine, accessibility settings, support of KhiCAS, external app, choose the multiplication sign, more steps for brightness, physics & chemistry constants in the toolbox, your name can be displayed in the settings, 32KB Python heap instead of 16KB,...
